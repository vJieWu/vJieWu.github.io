2.Excel读取操作

  • sheet_name

    默认：读取sheet0

    None：返回全部sheet，字典形式

    int：选择第几个sheet，从0开始

    str：选择sheet的名称

    list：需要读取的sheet可以为str或int

  • header & index_col

    header：指定某一行作为表头，默认数据区第0行为表头

    index_col：指定某一列作为行索引

  • usecols

    指定读取哪些列

    None：全部列（默认）

    str：’A,C’, ‘A,C:E’(A列+C~E列)

    int-list：列号列表

    str-list：列索引名列表

  • skiprows

    跳过行

    int：跳过前n行

    int_list：需要跳过的行号

  • names

    如果原文件不含表头，则可以使用names传入列名称列表，即表头，此时需要将header显式设置为header = None，否则第一行数据被表头占据不显示

  • dtype

    Type_name dtype
    浮点型 float16/float32/float64(默认)
    整型 int8/int16/int32/int64(默认)
    字符串 str(属于混合对象object)/string
    布尔 bool
    时间戳(纳秒) datetime64[ns]
    时间周期（Y/M/D) period[Y/M/D]
    python混合对象 object
    分类 category

    dtype作为字典输入，key为列索引的名称，value为dtype，这样指定每一列的数据类型

    category可以用于分类，如季节可分为春夏秋冬四季，指定这一列为category

  • parse_dates & date_parser

    parse_dates = False # 指定需要解析成日期的列

    data_parser = None # function,解析日期格式的函数

    tips：不包含中文的情况下，基本都可以直接解析，不需要设置解析日期的函数

    • parse_dates

      Ture: 尝试将索引类转化为日期类型，如果失败，则不转化，保持原始类型

      False：默认False，即不解析，如果在Excel中本来就是一个日期格式，不设置该参数也会解析为日期格式

      list: (int or str)将列表中的列号的列转化为日期类型，如[1,2,3]则将第1，2，3列的数据转化为日期格式

      2d-list: (int or str)每一行中的元素可分别代表年月日，[[1,2,3],[4,5,6]]将（1 2 3 ）（4 5 6）列，拼接为两个日期

      dic: 功能同2d-list，字典的key作为合并之后的新的列名称，如{‘日期’:[1,2,3]}，将123列拼接成日期，并命名为日期

    • data_parser

      • python lambda用法：定义一个函数

      • 举例：含有中文解析为日期类型 2020年6月27日

        首先设定日期所在列，再设置解析函数

  • na_values

    空值处理->NaN(Not a Number),或将指定数据解析为NaN

    list: 将整个表中的指定值转化为NaN，[‘a’, 0, ‘ ‘]->将a，0，空格都解析为NaN

    dic：指定列解析，{‘col1’:[‘a’, 0, ‘ ‘]}->将名为col1的一列中 a，0，空格都解析为NaN，或{1:[‘a’, 0, ‘ ‘]}->对第一列做操作

  • converts

    读取的时候对某一列做函数处理

    dic: {‘col1’: lambda x: x.strip(), 1: str.strip}, 将列名为col1的列中数据去除空格，以及第1列中的数据去除空格

  • index_col

    指定行索引使用哪些列

    index_col = 0，第0列作为行索引

    index_col = [0,1], 前两列作为联合索引

  • squeeze

    如果数据只有一列，是否返回一个Series

    False: 默认，仍然返回DataFrame

    True：返回Series

  • mangle_dupe_cols

    如果出现列重名了，进行重命名处理

    False: 重名则报错

    True：如果俩都叫Name，则读入之后一个叫Name，另一个叫Name.1（默认）

  • nrows

    读取的行数，int， 默认None，全部读取

  • thousands

    默认None，对于千位分隔符不处理

    thousands=’,’ : 将带有分隔符的数据解析为数字

  • convert_float

    默认True: 对于整个表尽可能将浮点数转化为整数，即1.0->1，一整列都能转化为Int，才会转化

    False: 全部浮点数读入，整型也会以浮点数形式读入
